動動手啊動動腦

By EmilyOwl 10 January 2021

如果沒有產出終將成為廢物

就在前年我沾沾自喜用 Hugo 架了 github page 之後我再也沒有更新過，直到今日想到還有這個地方時，github 傳來了這個消息

Basic authentication using a password to Git is deprecated and will soon no longer work.

真是謝啦，廢渣如我覺得 github desktop 好難用喔救命 🙀

本來是為了記錄自己在台灣 Gap Year 被包養 的過程，但沒有收入的日子容易焦慮、很難有穩定的產出，也因此遲遲沒有更新，在一年過後，雖然還不到財富自由，但至少做到了 減少工時、有基本收入 ，非常感激這一年來伸出援手的人們、也很感激老天的安排和善待。為了錢活下去，我捨棄了「追求對世界有正面意義的事物」這個執念，先在 Gap Year 各種嘗試中選擇了一個稍微有風險的但報酬率也是最高的方案，幸運的得以在半年後繼續敲著鍵盤更新部落格，但我並未忘記在清邁時許下的願望：想要讓世界因為我的存在變好一點點。而在那 Gap Year 一年時養成的寫工作日誌的習慣、強迫自己從每日所做的事情裡找出成長的信號其實幫助我甚多，於是回到這個地方，想繼續記錄成長的過程直到回清邁還願的那一天。

沒人料到 2020 年在武漢肺炎的肆虐下如此黯淡地結束，但也是因為死亡的威脅，我總覺得自己對於物質的慾望變少了、和自己的對話變誠實了，成功活到了 2021 年，世界即將變得更好還是更壞呢？ 讓我繼續參與下去吧！

Reinvent yourself. Time erodes every advantage. Evolve or die.

James Clear
